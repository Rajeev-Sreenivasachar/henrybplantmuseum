Website Development

code

How This Website Was Created

Built using Code.org Web Lab with HTML and CSS for Regionals and States competitions. Switched to Vercel using a GitHub repository for Nationals competition to implement JavaScript and advanced features. Used w3schools for testing our website for Nationals.

code.org → vercel.com → github.com → w3schools.com →
AI development

Henry AI

We developed Henry AI to help users with their questions about the museum, help adjust accesibility settings, and navigate through different pages in the website. We created our AI agent by embedding an API key from Google's AI Studio.

Google's AI Studio →
profile development

Museum Portal Authentication

We developed a full functioning Sign-in and Sign-out system called the Musuem Portal. We created and stored the account data using a Firebase Authentication Servers from Google.

Firebase Authentication →
newsletter development

Henry B Plant Museum Newsletter

We developed a monthly newsletter emails and a full functioning newslettter website for our museum. We used Beehiiv to embed, create, and send the newsletters.
