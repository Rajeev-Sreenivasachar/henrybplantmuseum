code

How This Website Was Created

Built using Code.org Web Lab with HTML and CSS for Regionals and States competitions. Switched to Vercel using a GitHub repository for Nationals competition to implement JavaScript and advanced features. Used w3schools for testing our website for Nationals.

code.org → vercel.com → github.com → w3schools.com →

Why This Website Was Created

We built website to be user-friendly, professional, and fully fuctional for all users, helping visitors learn more about our museum and our activities.
